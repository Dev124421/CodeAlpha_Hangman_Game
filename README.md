# CodeAlpha_Hangman_Game
Hangman game project

Goal : create a simple text-based Hangman Game where the player guesses a word one letter at a time. 
Key Concepts used : random , while loop , if-else , strings , lists . 
procedure : 
  1. A set of predefined words is provided to the system out of which any one word is selected .
  2. The user enter any letter and if it is present in the word , the letter is displayed in the word at its position acting as an hint for the word.
  3. If the user gives six incorrect letters before the word is guessed , the user lost the game and the word is displayed with a lost message .
